# TareaSO2
Parte 1 Implementacion de Barrera
Para compilar usar la siguiente linea dentro de la carpeta del archivo
gcc -o Barrera hebras.c barrera.c -pthread 
Y para ejecutar se debe introducir un parametro numerico correspondiente al numero de hebras
./Barrera 6    Ejecuta el programa simulando el trabajo de 6 hebras
